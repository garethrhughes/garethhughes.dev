# Decisions

Architectural Decision Records (ADRs) for garethhughes.dev. Each file follows the format `NNNN-short-kebab-case-title.md`.

These complement the flat `DECISIONS.md` log at the repo root. New decisions should be recorded here (and summarised in `DECISIONS.md`). Existing decisions from `DECISIONS.md` can be migrated here using the `decision-log` skill.

See the `decision-log` skill for the ADR format.
